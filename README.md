# Dejavu Project Page

This repository contains the source for the Dejavu project website:

- Title: `Dejavu: Towards Experience Feedback Learning for Embodied Intelligence`
- Venue: `CVPR 2026`
- Main page: `index.html`

## Local Preview

Because this is a static site, you can preview it with any simple HTTP server. For example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Notes

- The paper PDF is served from `assets/dejavu.pdf`.
- The website intentionally shows `Code (coming soon)` for now.
- Unused template assets and scripts were removed to keep the site lightweight.
